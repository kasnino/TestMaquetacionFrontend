Este es un Prototipo de Maquetacion Web y un poco de Logica de programacion con Vanilla Javascript..
Uso herramientas como Bootstrap para mejorar la paginacion, los iconos son de Awesome Icons, precompilador de SASS a travez de consola CMD..
La mayoria de las animaciones en CSS son realizadas a mano sin ayuda del framework con el fin de tener mayor control del resultado..
En este proyecto hay mucho codigo que se puede automatizar "Repetitivo" sacandole el verdadero potencial de SASS y la Libreria ReactJS (Componentes)..
Tiempo demorado en realizarlo 14 Horas varios intervalos.!!
